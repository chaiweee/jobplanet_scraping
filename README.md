# jobplanet_scraping
Using Python Selenium to get IT companies information from Jobplanet website   
셀레니움으로 잡플래닛 IT기업 정보 크롤링

## Steps:  
1. web scrape
2. write to csv
3. load data from csv for cleaning
4. save result to db
5. do simple eda
